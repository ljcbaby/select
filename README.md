# 抽签系统后端

## 项目介绍

本项目是一个抽签系统的后端，使用 Go 语言编写，使用 Gin 框架，使用 MySQL 数据库和 Redis 缓存。
  
创新实践课程作业，前端项目地址：[待补充](https://github.com)
